Summary of development:

The sports shop system is a NodeJS application, distributed via NPM which also have both: NPM and Bower dependencies. The application is built on top of:
  • ExpressJS 4
  • Mongoose
  • BackboneJS
  • Backbone-Forms in combination with Backbone-Deep-Model
  • Bootstrap3
  • MongoDB was used as the DB. MongoDB is a NoSQL DB, providing a wide range of benefits
